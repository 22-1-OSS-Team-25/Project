## **What is Uledger?**

ULedger는 CLI 기반의 빠르고 쉬운 사용이 가능한 오픈소스 가계부 프로그램입니다.

수입에 따른 지출을 체계적으로 관리하고, 소비 습관을 개선해 나가는데 도움을 줍니다.

## **It can ...**

- 수입과 지출을 한번에 관리할 수 있습니다.
- 입력된 데이터를 여러 조건으로 검색할 수 있습니다.
- 소비 유형과 동향을 파악할 수 있습니다.

이 외에도 다양한 기능을 제공합니다.
